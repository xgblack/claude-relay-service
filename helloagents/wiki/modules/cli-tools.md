# cli-tools

## 目的
提供命令行与脚本工具，支持安装、服务管理、数据迁移与调试。

## 模块概述
- **职责:** 通过 `cli/` 与 `scripts/` 提供安装、监控、迁移、定价更新等脚本
- **状态:** ✅稳定
- **最后更新:** 2025-11-27

## 规范

### 需求: 脚本化运维
**模块:** cli-tools
通过 npm 脚本与自定义脚本执行服务管理与数据处理。

#### 场景: 服务管理
- 运行 `npm run service:start` 等命令
- 启动或停止服务，输出统一日志

## 依赖
- Node.js 环境；部分脚本依赖 bash
